Browser Keyboard

Virtual keyboard extensoin for Firefox and Chrome.

View on GitHub

View Demo

Chrome alpha test

Features

convert physical key press to extensoin layout symbols

type text by on-screen keyboard

choose diferent languages behind system settings

hide/show on-screen keyboard however extension will continue to covert physical key press

turn on/off auto keyboard capturing

language list is downloaded from server on resetting enxention languages by AJAX

lowercase, uppercase, additional lowercase and additional uppercase symbols sets

popum menu:

switching on/off virtual keyboard

hide/show on-screen keyboard

change current language

hot keys:

ctrl+k — switching on/off virtual keyboard

shift

caps lock

ctrl + alt — shift-like switch symbols set to additional letters

ctrl + alt + space — caps-like switch symbols set to additional letters

shift + space — change language layout to next

also able to combinate hot keys, ex: shift + ctrl + alt — switch symbols set to additional uppercase letters

functional buttons on on-screen keyboard:

close — turn off virtual keyboard

minimize — hide on-screen keyboard

shift

caps lock

addit — shift-like switch symbols set to additional letters

additLong — caps-like switch symbols set to additional letters

space

backspace

delete

enter

next language

on-screen keyboard is dragable
